A simple React.js application with a Node.js/Express backend to search and view student details using a search bar with lazy loading and debounce.

**How to Run**
Frontend- 
- cd frontend
- npm install
- npm run dev

Backend- 
- cd backend
- npm install
- npm start

**features**
- Debounced search input
- Lazy loading of results
- Click to view detailed student info
- Backend API using Express and local JSON file


**Dependencies**: react, react-dom, axios
